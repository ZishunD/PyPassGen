# 🔐 PyPassGen - 密码生成器 (CLI + Web)

一个使用 Python 编写的命令行 + Web UI 密码生成器，支持自定义长度、复杂度，部署到 Render 可用。

## 🚀 技术栈

- Python + Flask
- TailwindCSS (Web UI)
- Docker
- Render (云部署)

## 🧪 功能

- ✅ 自定义密码长度
- ✅ 控制是否包含数字/符号
- ✅ Web UI 展示最新 5 个生成历史
- ✅ Render 免费部署

## 📦 使用

```bash
git clone https://github.com/yourname/pypassgen.git
cd pypassgen
pip install -r requirements.txt
python app.py
